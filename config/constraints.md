# Ideation Constraints

## Domain Focus
Developer tools, SaaS infrastructure, and internal tooling for engineering teams.

## Technical Boundaries
- Must be buildable as a web app or CLI tool
- No hardware, no IoT, no mobile-only apps
- Prefer solutions that can run on a single server or serverless

## Market Preferences
- B2B only
- Target: small-to-mid engineering teams (5-50 developers)
- Price range: $20-500/month per team
- Must have a plausible path to $10K MRR within 12 months

## Anti-Patterns
- No crypto/blockchain/web3
- No social media platforms
- No AI wrappers that just call an LLM API with a prompt
- No Jira clones or generic project management tools

## Current Interests
- Developer experience and productivity gaps
- Observability blind spots
- CI/CD pain points
- Cost management for cloud infrastructure
